This is a demo of an SMS-enabled Web Application, designed for users to take advantage of New York City's Data Mine initiative. With vast majority of people not having browser access on the go, we decided to design an SMS application that allows users submit 311 complaints and query several City Agency datasets without needing a computer or a fancy phone.

To experience the demo, you can either SMS to nycsamosa@gmail.com from your cell phone or use the following interface. Type a query in this box or simply click the links below to view sample search results in the cell phone screen below. Try the same from any cell phone. After you get a list of results like Events 1-5, you can either send more for next batch or send more plus the number of event you want see details of: more 4 to see details of the 4th event.

One of the key efforts is to allow users to input natural phrases, without many restrictions by any special syntax. You still need to have correct order of elements, so it's not perfect yet, but getting there :)

311 Reports

After submitting a 311 complaint, you can go to the 311 complaint Dashboard: http://nycsamosa.appspot.com/dashboard and see where it fits on the NYC map along with other complaints. Report by typing issue at location. See examples above!

Traffic Updates

So, what if you drive through Manhattan every other week, to a different address in a different borough. Every time you turn on the radio for traffic updates, they talk about some place else... You want to know what's on your route, not somewhere else! You don't have a full blown browser in your phone to navigate through tiny controls and navigate to some fancy web site.  You just want to know what to expect.
Query by typing traffic location1 to location2. See examples above!

Location-based Search

Search for Cafes, Parking garages, Wifi hotspots or Laundromats. The search with return five closest hits, based on geographic location provided.
Query by typing cafe at location. See examples above!

Event Search

Upcoming NYC community events provided via RSS feed are available via date-based search.
Send events date. See examples above!

Alternate Parking Calendar

Future Alternate Parking regulation suspension dates are available by simply sending altparking. See examples above!

So now you can use any simple cell phone, and benefit from the same, up-to-date information as those sitting by a PC and walking around with iPhones (not that they are not cool). So it comes down to the associated costs or SMS'ing, which are dependent on the plan with your provider.
This is a prototype. Long-term we envision users sending messages to 311 using a gateway. This significantly improves Quality of Service because we will bypass the email part completely!
